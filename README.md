# Создание интерфейса авторизации и регистрации для системы
Данная практическая работа посвящена разработке клиентской части информационной системы "Система ревью кода" на платформе WPF (Windows Presentation Foundation) с использованием языка C# и базы данных MS SQL Server.

### Выполненные задачи:
1.  **Разработано WPF-приложение** со следующими окнами:
    *   `LoginPage` — окно авторизации пользователей с валидацией ввода.
    *   `RegistrationPage` — окно регистрации новых сотрудников с проверкой сложности пароля и уникальности email.
    *   `AdminPage` — панель администратора со статистикой системы (пользователи, проекты, ревью) и таблицей последних запросов на ревью.
    *   `MainWindow` — главное окно для обычного пользователя.
2.  **Реализована интеграция с базой данных** через Entity Framework (Database First / EDMX):
    *   Авторизация и регистрация пользователей с проверкой данных в таблице `users`.
    *   Разграничение прав доступа: пользователи с ролями "Administrator" или "Manager" перенаправляются в панель администратора, остальные — в главное окно.
    *   Динамическая загрузка статистики и списка последних ревью из БД.
3.  **Внедрен фирменный стиль** в соответствии с техническим заданием:
    *   Цветовая палитра IDE: темный фон (#263238), бирюзовый акцент (#00BCD4), розовый для ошибок (#FF4081).
    *   Использование моноширинного шрифта Consolas для полей ввода.
    *   Централизованное управление стилями в файле `App.xaml`.
4.  **Настроена навигация** между окнами и корректная обработка исключений при подключении к БД.

### Технологии:
*   **Платформа:** WPF (.NET Framework / .NET Core)
*   **Язык:** C#
*   **ORM:** Entity Framework 6 (Database First)
*   **СУБД:** Microsoft SQL Server
*   **Стилизация:** XAML Resources & Styles
